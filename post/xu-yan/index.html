序言

2021-04-04 2 min read # 序言 # 读书 # 博尔赫斯 # 夜晚的潜水艇 # 故事

“我反复画过一张画。深蓝色的背景中央，有一片更深的蓝。有人说像叶子，有人说像眼睛，像海里的鲸鱼。人们猜想其中的隐喻。”

读过很多序言，几乎每本书都有。

印象深刻的比如《夜晚的潜水艇》：博尔赫斯在海上邮轮往大西洋丢下一枚金币引来一段奇事，后世一位痴迷博氏以致连其中文版译者王永年所用钢笔都花重金购得的西洋富商，不计成本聘请一名陈姓海洋学家，带队驾驶潜艇在海下用金属探测器寻找博尔赫斯在书中所记叙的那枚带着指尖余温沉向海底的金币。

富商一生痴迷于看海底潜水打捞录像直至最后一刻。

海洋学家之孙陈透纳（借作者之姓和画家透纳为名的虚构人物）在孩童时期具有将一切现实化为想象材料的能力，最大乐趣是在上课和睡觉时进入自己的梦境驾驶自己发明的潜水艇在海底探险，后来父母不断规劝遂认真学习，但永远失去了那种想象力，考入大学，进设计公司，最终成为画家，在遗书中他坦言后来所有瑰丽的画作都是画的孩童时期那番海底历险亲眼所见的一切，画完了，就再没了。

陈透纳遗书的最后一段，交代了继承事宜后，他写道：“我反复画过一张画。深蓝色的背景中央，有一片更深的蓝。有人说像叶子，有人说像眼睛，像海里的鲸鱼。人们猜想其中的隐喻。其实没有任何含义，那是一艘潜水艇。我的潜水艇。它行驶在永恒的夜晚。它将永远，永远地悬停在我深蓝色的梦中。”
